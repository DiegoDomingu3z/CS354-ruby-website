Description & History

PHP, short for "PHP: Hypertext Preprocessor", is a widely used web developement language

 

Notable Characteristics:

- Interpreted server-side scripting language

- Multi-paradigm

- Dynamically typed

- Executed on the server, no code is sent to client

- Used for dynamic webpages

- Free to use under the PHP License

 

History

- Developement began in 1993 by Danish-Canadian programmer Rasmus Lerdorf

- Invented so he could maintain his personal website easier

- Had basic HTML and database communication functionality

- Initially called PHP/FI, it stood for "Personal Home Page/Forms Interpreter"

- Was never intended to become a programming langauge, just kept getting features added to it

- First public release happened in 1995

- PHP/FI 2 released in 1997

- PHP's third release is where it become known as just "PHP", which happened in 1998

- Zend Engine is a rewrite of PHP's core, which powers many future releases of PHP

- PHP 8.x is the family of versions in development today today
